# Book Genre Classification using Machine Learning

Predicting book genres from structured Goodreads metadata using 10 ML models.

## Dataset
Goodreads Books with Genres — Kaggle (5,283 books, 6 genres)

## Models Used
**Base:** Logistic Regression, KNN, SVM, Decision Tree, Naive Bayes
**Ensemble:** Random Forest, XGBoost, Gradient Boosting, AdaBoost, Bagging

## Best Model
Random Forest — 42.10% accuracy, best balance of speed and performance

## How to Run
1. Open `ML_MiniProject.ipynb` in Google Colab
2. Upload `booksgenre.csv`
3. Run all cells in order

## Results
| Model | Accuracy |
|---|---|
| SVM (RBF) | 42.29% |
| Random Forest | 42.10% |
| XGBoost | 40.30% |
| Naive Bayes | 25.45% |
